## Capture Tips

- Use a clean browser profile or clear `localStorage` before capturing the initial honest refusal state.
- Capture the second assistant answer after approving `profit / 利润`.
- Keep the browser width close to a recruiter-friendly desktop screenshot, such as 1440px.
- Use the same light theme for all screenshots.
